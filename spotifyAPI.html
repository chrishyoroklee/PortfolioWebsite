Python Software For Spotify API

PlaylistSongFeatures file allows you to input a Spotify playlist URL, and get back a selection of audio features - Danceability, Duration, Energy, Valence, Tempo, Speechiness, Mode, Loudness, Liveness, Key, Instrumentalness.

These audio features are set down by Spotify, and these values are perfect for creating machine learning models or conducting various research.

Inside the SpotifyAPI github project, there is also a file named individualSong, where users can find these same audio features for individual songs on Spotify by using Spotify song URL as the input.
