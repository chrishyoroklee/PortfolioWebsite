Musical Circles

This was designed as an interactive tool where anyone without any music knowledge or experience can create music.
I first started this project with the 2D version in p5.js, and moved onto creating the 3D version by using three.js and node.js

Python Software for Accessing Spotify API

This is a python code written to access the Spotify API, and retrieve various metadata about audio features. PlaylistSongFeatures file allows you to input a Spotify playlist URL, and get back a selection of audio features - Danceability, Duration, Energy, Valence, Tempo, Speechiness, Mode, Loudness, Liveness, Key, Instrumentalness. Then, these audio feature data can be used to create machine learning models.
